## Description
 The repro APIs for lotus's eat mobile application project.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode (develop)
$ npm run dev

# production mode
$ npm run build && npm run start:prod
```

## Test

```bash
# unit tests and show coverage
$ npm run test
```
